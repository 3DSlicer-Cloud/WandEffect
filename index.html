Welcome to WandEffect

This is a plug in example for 3D Slicer (see http://www.slicer.org).

What's the idea?

The Slicer 4 Editor module is extensible by writing python scripts like the one in this repository.

The superclasses handle a lot of things about the way the volume data is managed so that the plugin can respond to mouse moves and clicks to implement interactive segmentation tools.

See the documentation for this project here:

http://www.slicer.org/slicerWiki/index.php/Documentation/4.0/EditorExtension
